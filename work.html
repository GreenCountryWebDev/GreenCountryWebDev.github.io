Weather Or Knot

This was my final project at Holberton. I created a desktop weather app using React and Tailwind CSS. The app utilizes the OpenWeather API to display current weather conditions and a five day forecast. The second half of this project was a recreation of the classic game Snake. I created the game using React and CSS. The landing page was created using React and CSS. Click on the image to visit the page.
